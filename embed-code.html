Responsive Embed (All Devices)

Use this code for a fully responsive embed that adapts to any screen size:

<!-- Responsive Portfolio Embed --> <div style="position: relative; padding-bottom: 75%; height: 0; overflow: hidden; max-width: 1200px; margin: 2rem auto;"> <iframe src="https://YOUR-DOMAIN.com/tpe-detail.html" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: 2px solid #20B2AA; border-radius: 15px;" frameborder="0" title="Portfolio Project"> </iframe> </div>
Note: Replace "tpe-detail.html" with any portfolio detail page (tpe-detail.html, boxes-detail.html, etc.)

WordPress Compatible

For WordPress sites, use the HTML block or Custom HTML widget:

<!-- WordPress HTML Block --> <div class="portfolio-embed-container"> <iframe src="https://YOUR-DOMAIN.com/tpe-detail.html" width="100%" height="800" frameborder="0" loading="lazy" style="border: 2px solid #20B2AA; border-radius: 15px;" title="Portfolio Project"> </iframe> </div>

Dynamic JavaScript Embed

Use this JavaScript code to dynamically embed portfolios with a simple button click:

<!-- HTML Button --> <button onclick="loadPortfolio('tpe-detail.html')">View The People's Economy</button> <div id="portfolio-container"></div> <!-- JavaScript --> <script> function loadPortfolio(pageUrl) { const container = document.getElementById('portfolio-container'); container.innerHTML = ` <iframe src="https://YOUR-DOMAIN.com/${pageUrl}" width="100%" height="800" frameborder="0" style="border: 2px solid #20B2AA; border-radius: 15px; max-width: 1200px; display: block; margin: 2rem auto;" title="Portfolio Project"> </iframe> `; } </script>

Modal/Popup Style Embed

Open portfolio details in a modal overlay for a modern user experience:

<!-- Modal HTML --> <button onclick="openPortfolioModal('tpe-detail.html')">View Project</button> <div id="portfolioModal" style="display:none; position:fixed; top:0; left:0; width:100%; height:100%; background:rgba(0,0,0,0.9); z-index:9999; padding:2rem;"> <button onclick="closePortfolioModal()" style="position:absolute; top:2rem; right:2rem; background:#20B2AA; color:white; border:none; padding:0.5rem 1rem; border-radius:5px; cursor:pointer; font-size:1.5rem;">Ã—</button> <iframe id="modalIframe" width="100%" height="100%" frameborder="0" style="border-radius:15px;"></iframe> </div> <script> function openPortfolioModal(pageUrl) { document.getElementById('portfolioModal').style.display = 'block'; document.getElementById('modalIframe').src = 'https://YOUR-DOMAIN.com/' + pageUrl; } function closePortfolioModal() { document.getElementById('portfolioModal').style.display = 'none'; document.getElementById('modalIframe').src = ''; } </script>
Important: Replace "YOUR-DOMAIN.com" with your actual website domain where these HTML files are hosted. Make sure all detail HTML files (tpe-detail.html, boxes-detail.html, etc.) are uploaded to your web server.
